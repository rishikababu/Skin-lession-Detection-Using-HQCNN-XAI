# REAL-TIME SKIN LESION ANALYSIS AND SKIN CANCER DETECTION USING HQCNN AND EXPLAINABLE AI

## 📌 Project Overview

This project is a web-based intelligent medical diagnosis system that uses a Hybrid Quantum Convolutional Neural Network (HQCNN) and Explainable AI (XAI) for skin lesion analysis and skin cancer detection. It is designed to assist in the early identification of skin diseases by combining deep learning, quantum-inspired computing, and AI-driven decision support.

**Note:** This is an academic/ demo project designed to demonstrate skin lesion analysis and skin cancer detection using HQCNN and Explainable AI. The results are for educational and demonstration purposes only and should not be used as a replacement for professional medical advice, diagnosis, or treatment.
Also i have uploaded the main codes of this project.

## ✨ Key Features

* 🔬 Real-Time Skin Lesion Analysis
* 🧠 Hybrid Quantum Convolutional Neural Network (HQCNN)
* 🎯 Detects and Classifies 7 Different Types of Skin Lessions
* 🔥 Explainable AI using Grad-CAM Heatmaps
* 📊 Confidence Score Evaluation
* ⚠️ Severity Assessment and Medical Recommendations
* 📄 Automated PDF Medical Report Generation
* 📧 Email Notification System
* 👤 Patient Dashboard with Search and History Tracking
* 👨‍⚕️ Doctor Review and Approval System
* 👨‍💼 Admin Analytics Dashboard
* ⚛️ Quantum Processing Visualization
* 🔐 Role-Based Authentication and Authorization
* 💾 SQLite Database Integration

## ⚛️ Quantum Computing Component

* Quantum simulation was implemented in the HQCNN model to explore hybrid quantum-classical learning approaches.
* Quantum circuits were simulated and integrated with CNN-based feature extraction for enhanced skin lesion classification.
* The project includes a Quantum Visualization module to demonstrate the flow of quantum-inspired processing within the diagnosis system.
